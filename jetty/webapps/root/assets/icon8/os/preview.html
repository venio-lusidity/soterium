CSS

Unzip your copy of web-font into your project. In the <head> of your html add link to CSS file:

<link rel="stylesheet" href="file/to/your-font/css/styles.min.css">

In html you may insert icons like that:

<i class="icons8-android-os"></i>
<i data-icons8="&#xf10e;"></i>

Basic LESS

Unzip your copy of web-font into your project. In your LESS file @import LESS styles from web-font:

@import "file/to/your-font/less/styles.less";

Advanced LESS

@import "file/to/your-font/less/variables.less";
@import "file/to/your-font/less/core.less";
@import "file/to/your-font/less/paths.less";
@import "file/to/your-font/less/icons.less";

// make glyph alias with custom behavior
.some-new-icon:after { // <i class="some-new-icon"></i> is the same as <i class="icons8-android-os"></i>
  .icons8-font();
  font-size: 2em;
  content: @icons8-var-android-os;
}

Basic SCSS

Unzip your copy of web-font into your project. In your SCSS file @import SCSS styles from web-font:

@import "file/to/your-font/scss/styles.scss";

Advanced SCSS

@import "file/to/your-font/scss/variables";
@import "file/to/your-font/scss/core";
@import "file/to/your-font/scss/paths";
@import "file/to/your-font/scss/icons";

// make glyph alias with custom behavior
.some-new-icon:after { // <i class="some-new-icon"></i> is the same as <i class="icons8-android-os"></i>
  @include icons8-font();
  font-size: 2em;
  content: $icons8-var-android-os;
}
